Standard Suite

Common classes and commands for most applications.

Commands

  • close -- Close an object.
    • reference -- the object for the command
    • [saving ask / no / yes] -- Specifies whether changes should be saved before closing.
    • [saving in alias] -- The file in which to save the object.
  • count -- Return the number of elements of a particular class within an object.
    • reference -- the object for the command
    • [each type class] -- The class of objects to be counted.
    • Result: integer
  • delete -- Delete an object.
    • reference -- the object for the command
  • duplicate -- Copy object(s) and put the copies at a new location.
    • reference -- the object for the command
    • [to location reference] -- The location for the new object(s).
    • [with properties record] -- Properties to be set in the new duplicated object(s).
  • exists -- Verify if an object exists.
    • reference -- the object for the command
    • Result: boolean
  • get -- Get the data for an object.
    • reference -- the object for the command
    • Result: anything
  • make -- Make a new object.
    • new type class -- The class of the new object.
    • [at location reference] -- The location at which to insert the object.
    • [with data anything] -- The initial data for the object.
    • [with properties record] -- The initial values for properties of the object.
    • Result: reference
  • move -- Move object(s) to a new location.
    • reference -- the object for the command
    • to location reference -- The new location for the object(s).
  • open -- Open an object.
    • alias -- The file(s) to be opened.
    • Result: document
  • print -- Print an object.
    • alias -- The file(s) or document(s) to be printed.
    • [print dialog boolean] -- Should the application show the Print dialog?
    • [with properties print settings] -- the print settings
  • quit -- Quit an application.
    • [saving ask / no / yes] -- Specifies whether changes should be saved before quitting.
  • save -- Save an object.
    • reference -- the object for the command
    • [as Unicode text] -- The file type in which to save the data.
    • [in alias] -- The file in which to save the object.
  • set -- Set an object's data.
    • reference -- the object for the command
    • to anything -- The new value.

OmniGraffle

Omni Graffle Script Suite

Commands

  • assemble -- Group graphics
    • reference -- the object for the command
    • [subgraph boolean] -- A subgraph with a background
    • [table shape list] -- The shape of a tabular group
    • Result: group
  • connect -- Draw a line between graphics
    • reference -- the object for the command
    • to anything -- Graphic or graphics to connect to.
    • [with properties record] -- The initial values for properties of the line.
    • Result: line
  • flip -- Flip graphics
    • reference -- the object for the command
    • over horizontally / vertically -- Orientation to flip the graphic around
  • import -- Open or import a file. The command can optionally specify settings to use when converting other file types to Graffle documents.
    • reference -- the object for the command
    • [categories boolean] -- Include class categories when importing source code?
    • [frameworks boolean] -- Include frameworks when importing source code?
    • [instance variable types boolean] -- Include the types of instance variables when importing source code?
    • [instance variables boolean] -- Include instance variables when importing source code?
    • [interaction allowed boolean] -- Is the application allowed to prompt the user for missing import settings? (If this is set to false, then Graffle will return an error rather than prompting the user for missing settings.)
    • [mapping record] -- A record or list associating outline column titles with template text field names
    • [method signatures boolean] -- Include the type signatures of methods when importing source code?
    • [methods boolean] -- Include method names when importing source code?
    • [outline template Unicode text] -- The name or path of the template to use for importing an outline
    • [protocols boolean] -- Include protocol definitions when importing source code?
    • Result: document
  • layout -- Layout some graphics, or a whole document using the document's Layout Info.
    • reference -- the object for the command
  • page adjust -- Change the number of pages to fit the current graphics
    • reference -- the object for the command
  • slide -- Slide graphics by a vector amount
    • reference -- the object for the command
    • by point -- The x and y amounts to shift the graphics
